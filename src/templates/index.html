Intense language learning

Learning a language in 3-6 months teaches you that you can do things that others cannot imagine.
Also, it is a necessary requirement for any job outside Tunisia.

The keys are total immersion and focus.
We help you get to a B1 level in German in 3 months, if you make it a priority.
This is uncomfortable.

Focused skill-building

There are core skills that are critically relevant for a data scientist or data engineer.
Two very basic examples are the use of git as a source control tool or SQL as a database query language.
You can learn these over a weekend and benefit from them for the rest of your life.

Commit to doing the basics right. The rest will come in time.
We help you apply these basics in actual work assignments.

Seminars on execution

Because we received a certificate from somewhere, we believe we know things.

So no, we don't know things.

If you're interested, we help you get better at getting things done.
This is uncomfortable.

Why?

While cutting-edge competence is valuable, most companies cannot integrate candidates with such skillsets.

What everyone needs is co-workers that get things done.
